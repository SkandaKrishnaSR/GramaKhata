🏡 GramaKhata

A simple Android app for managing village/property records digitally using Kotlin and Jetpack Compose.

📱 About the App

GramaKhata helps users store and manage village property records digitally.
The app works completely offline and provides a simple interface for adding, updating, searching, and deleting records.

🎯 Problem Statement

In many rural areas, records are still maintained manually using paper files.
This can lead to:

Missing records
Duplicate entries
Human errors
Difficult searching
Time-consuming management

GramaKhata solves this problem with a simple digital record management system.
